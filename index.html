Square - an euclidean percussion sequencer for musical experimentation on the Game Boy.

Requirements: Gameboy DMG or Emulator


Explore euclidean drum patterns on your Gameboy.

Features

* Three drum sounds
* Four in-session save slots


The rom and source code of Square can be checked out at GitHub.

Manual

COMBOS

SELECT + UP = SELECT DRUM
SELECT + DOWN = SELECT MODE (MASTER/SYNC)

FOR SELECTED DRUM

UP/DOWN = INCREASE/DECREASE STEPS
RIGHT/LEFT = INCREASE/DECREASE PULSES
A + UP/DOWN = INCREASE/DECREASE OFF-STEP
A + RIGHT/LEFT = INCREASE/DECREASE SWING

SAVING

B + UP/DOWN = SELECT SLOT
B + RIGHT = SAVE TO SLOT
B + LEFT = LOAD FROM SLOT
